# OCR_examples
OCR_examples

EasyOCR approach is used on site_header.png file. The result is saved in site_header_recognized.png file.

Keras-OCR approach is used on cocacola_logo.png file. The result is saved in cocacola_logo_recognized.png file.

OpenCV approach is used on google_logo.png file. The result is given to stdout, it was 'Google'.
Source and binary files are provided.

To use, change path to your file. CPP code requires recompiling.
